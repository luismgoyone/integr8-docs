# Reporting made easy

Details Reports giving you traffic insights, for GEO location, Session Times, Device Distribution, Fraud Reports and more. All Data is exportable and available through API.

 ![Click Report Screenshot](https://24metrics.getoutline.com/api/attachments.redirect?id=81021544-f62b-42db-b997-bb16bcae4885)

Integr8's platform offers a comprehensive suite of reporting and analytics tools designed to provide in-depth insights into your marketing campaigns. Here's an overview of the key features:

## [Overview Reports](https://integr8.co/docs/articles/overview-reports/)

The Overview Reports deliver a detailed analysis of click and conversion data, segmented by parameters such as Advertiser, Publisher, Offer, and SubID. The interactive dashboard presents key metrics and fraud profiles, enabling users to filter and group results for customized insights.

## [Traffic Insights](https://integr8.co/docs/articles/traffic-insights/)

Traffic Insights is a real-time analysis tool that offers an immediate overview of critical data points, including clicks, impressions, conversions, and events. Key features encompass fraud distribution, device and country breakdowns, and top IPs and fingerprints, facilitating swift and informed decision-making.

## [Click Reports](https://integr8.co/docs/articles/click-reports/)

The Click Report feature allows users to view and export detailed click data from their campaigns. Data can be grouped and filtered by various parameters such as publisher, offer, sub ID, user agent, IP address, browser language, status, and rejection reasons. The user interface supports up to 2,000 clicks, while exports can handle up to 2 million clicks. Customizable columns and CSV export options enhance data analysis capabilities.

## [Conversion Reports](https://integr8.co/docs/articles/conversion-reports/)

The Conversion Report enables users to access and export raw conversion data. Filtering options include publisher, offer, sub ID, user agent, IP address, browser language, status, and rejection reasons. Users can customize visible columns and export reports as CSV files for further analysis.

## [Shared Reports](https://integr8.co/docs/articles/shared-reports/)

Shared Reports facilitate seamless collaboration by allowing users to generate public links to specific reports, such as Overview, Click, Conversion, and Event reports. This feature enables partners to access pertinent data without requiring an AdSecurity account.

## [Export Reports](https://integr8.co/docs/articles/export-reports/)

Integr8's platform allows for the export of various reports—including Overview, Click, Conversion, Events, and Postback Logs—to CSV files. Users can select specific time frames, customize columns, apply filters, and export data for offline analysis or sharing with team members.

## [Postback Logs](https://integr8.co/docs/articles/postback-logs/)

The Postback Logs feature provides detailed records of affiliate postbacks, including data on firing time, transaction IDs, offer and publisher IDs, postback URLs, status codes, and any errors. This tool is essential for analyzing postback activities and troubleshooting issues.

## [Server Logs](https://integr8.co/docs/articles/server-logs/)

Server Logs offer comprehensive records of all server requests, capturing both successful and rejected transactions. The logs are divided into Attribution Logs, Conversion Logs, and Conversion Rejections, providing a granular view of server interactions for thorough analysis.

## [Download Center](https://integr8.co/docs/articles/download-center/)

The Download Center serves as a repository for all exported reports, retaining them for 90 days. Users can access and download reports such as Overview, Clicks, Conversions, and Events, facilitating easy retrieval and sharing of data.

By leveraging these robust reporting and analytics tools, Integr8 empowers users to gain actionable insights, optimize campaign performance, and enhance decision-making processes.
