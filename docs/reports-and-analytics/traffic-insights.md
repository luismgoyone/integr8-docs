## Introduction

Traffic Insights is a powerful tool for real-time traffic analysis, providing a quick overview of key data points to facilitate fast and informed decision-making.

 ![show columns](https://24metrics.getoutline.com/api/attachments.redirect?id=69c00e23-1a31-41a5-b57e-9d4be7c07482)

## Activation

The functions is accessible through the Overview Report. To enable this feature, follow these steps:


1. Navigate to the **Reports** -> ***Overview Report***Â section.
2. Click the ***Edit Columns*** icon.
3. Toggle the ***Traffic Insights*** option to "**On**."

Once activated, the "*Show Details"* button becomes visible in reports. Clicking it opens a popup window with essential data for rapid analysis.

 ![traffic insights](https://24metrics.getoutline.com/api/attachments.redirect?id=cd1435d0-449b-4911-93af-c2949f6795bd)

## Key Features

Traffic Insights provides a comprehensive snapshot of traffic performance, including clicks, impressions, conversions, and events. Traffic Insights equips users with the data they need for efficient monitoring and optimizations of traffic performance. Key features include:

**Fraud Distribution**: A global summary of rejection reasons.   ![Traffic Insights - Fraud Distribution](https://24metrics.getoutline.com/api/attachments.redirect?id=c12c4fca-92fd-4586-bbf1-1f0a5dea0c25) **Fraud Histogram**: Hourly visualization of fraud distribution.

 ![Traffic Insights - Fraud Histogram](https://24metrics.getoutline.com/api/attachments.redirect?id=ce3bddc7-d092-4712-a12e-412c32fc02cd) **Filter Group Histogram**: Insights into filter group usage per hour.
