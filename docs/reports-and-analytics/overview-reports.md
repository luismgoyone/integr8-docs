## Introduction

There are two types of overview reports:


1. **Overview Report** (Global)
2. **Events Overview Report**

The global Overview Report offer a global analysis of click, impression and conversion data, providing insights by Advertiser, Publisher, Offer, and SubID. With its interactive dashboard, users can easily access key metrics and fraud profiles in a centralized view.

The Events Overview Report sumes up the total number of events recorded on a specific offer within a determined time frame.

### Accessing the Reports

* *Reports* -> ***Overview Report*** (Global) to view and interact with global data.
* *Reports* -> ***Events Overview*** to analyse data related to conversion/Events only

## Key Features

* **Date Picker**: select specific date or define custom time range.
* **Filtering and Grouping**: apply filters and group results by parameters like Advertiser, Publisher, Offer, or SubID to tailor your insights.
* **Column** **Customization**: customize visible columns to display only the data that matters to your analysis.
* **Custom** **View**: define the format of a report and save it as a View to be used with one click each time you need it.
* **Exporting** **Data**: export report results to a CSV file for detailed offline analysis or record-keeping.
* **Shared** **Reports**: onfigure a report with selected data and generate a public link for seamless affiliate access. Affiliates can view the results without requiring an AdSecurity account. This feature-rich report simplifies data analysis, improves collaboration, and ensures actionable insights are always within reach.
