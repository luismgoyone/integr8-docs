## Introduction

Export reports feature in Integr8 allows the possibility to download the data to a CSV file so that can be analyzed off line as well or shared with other team members and departments.

[export reports](/api/attachments.redirect?id=950f539b-f36b-45a3-9d00-b01c1c507035)

## Reports to export

The logs available for export are:Overview report

* Click Report
* Events / Conversions
* Postback Logs
* Daily Reports
* Impressions Reports
* SmartLink Reports

## CSV export setup

Navigate to the desired report and follow the next steps:

* Select the time frame for the exported results
* Deactivate unnecessary columns to avoid overload the report
* Use FILTERS to narrow data down to the specific need
* Click Export
* Go to Reports -> [Download Center](https://integr8.co/docs/download-center/ "Download Center Guide") to download the CSV file

 ![csv reports](https://24metrics.getoutline.com/api/attachments.redirect?id=5f357848-507b-4a15-b823-b19610968028)

## Note

Depending on the exported report size, the CSV file will not be instantly available for download. Once the process is complete, the downloadable CSV will be available in the **Download Center** section.
