## Introduction

The Integr8 tracking platform offers powerful data analytics capabilities, enabling users to filter and group results by virtually any data point. The platform provides a variety of reporting tools, including click logs, postback and server logs, shared reports and more.

The conversion and events report feature allows users to access and export raw conversion data for detailed analysis.

 ![conversions report](https://24metrics.getoutline.com/api/attachments.redirect?id=d90e0ce0-3f5d-45c3-86d0-1bfc8b7d012a)

## Key Features

### Advanced Filtering

Filter conversion and events by parameters such as publisher, offer, sub ID, user agent, IP address, browser language, status, rejection reasons and any other data point available.

### Customizable Columns

Personalize your report by enabling or disabling specific columns using the Edit Columns button.

### Custom View

Define a report view and save it for future use with one click.

### Data Export

Export conversion reports as CSV files and easily download them from the Download Centre. Integr8's Conversion and Events Report provides users with precise insights into campaign performance, empowering effective monitoring and data-driven decision-making.

### Share Events Report

Filter the data you want to share and generate a publisc link you can share with selected partners for real time reporting and analysis.
