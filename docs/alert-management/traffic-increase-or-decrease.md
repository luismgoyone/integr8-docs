## Introduction

The Traffic Increase and Decrease alerts will trigger notifications when clicks or conversions volumes are lower or higher then the percentage threshold set. The time period for the checks can be set 2, 4 or 6 days in the past to be compared with "yesterday" volumes.

## Setup Instructions

* Navigate to Alerts Management
* Click the +Create
* Follow the setup wizard
* Save

Note: only one notification can be set for a specific set of rules. For more different checks, create a new integration for each case.
