## Introduction

The Organization section in Integr8 includeds all the details of an account. This section can be edited and updated with new information about the company, billing details, users and permissions settings, API access and white lable setup for affiliate signup.

## Structure

The Organization includes the following sections:


1. ***User Management***
   * [Users](https://integr8.co/docs/articles/user-management/ "User Setup")
   * [User Roles](https://integr8.co/docs/articles/user-management/ "User rolses and permissions")
   * API Keys
2. ***Settings***
   * General settings
   * [Invoice Details](https://integr8.co/docs/articles/billing/ "Invoicing information")
   * [Card Payment](https://integr8.co/docs/articles/billing/ "Payment settings")
   * [Partner Login](https://integr8.co/docs/articles/affiliate-dashboard/ "Affiliate dashboard")

### General Settings

This section contains all the information of your organization and plan pricing and details.

### API Keys

Create a custom API Key by following the next steps:

* Navigate to Organization > User Management
* Select the ***API Keys*** TAB
* Click on ***+Create*** button to generate a new key
* Define the permissions
* Click ***Save***
