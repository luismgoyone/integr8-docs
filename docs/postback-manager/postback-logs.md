## Introduction

Postback logs is an useful tool to check and analyze the data related to the affiliate postbacks fired or rejected.

## Data

The logs will display data related to:

* Success Status: the failed postbacks will show the reason in the Error column
* Postback Name
* Postback Type
* Offer ID
* Publisher ID
* Event ID
* Transaction ID
* Response Code
* URL
* Error detail (when applied)
* Server Response

 ![affiliate postback logs](https://24metrics.getoutline.com/api/attachments.redirect?id=6ab82eb4-2e9b-494f-83e1-f83349875c85)

## Key Feature

The postback logs results can be easily filtered by any value that is or isn't an exact match or it's contain within any of the data available in the columns listed previously.
